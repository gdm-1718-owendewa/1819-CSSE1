- Wat is een dataformaat?

    Een cruciaal onderdeel van een communicatieprotocol waarmee het ontvangende apparaat op een logische manier kan bepalen wat er met de gegevens moet gebeuren en hoe het moet worden gedaan.

- Welke dataformaten zijn er?
    (meest gebruikte)

    Exchanging data 
        .JSON JavaScript Object Notation, is a standardized data format. JSON uses human-readable text in the form of data objects that consist of one or more attributes with corresponding values. It is used for data exchange between server and web application, as an alternative to XML.

        .CSV A comma-separated file, or CSV file, in English comma-separated values, is a specification for table files. The term should not be taken too literally, because variants, such as a semicolon as a separator, are also called so.

        .XML Extensible Markup Language Extensible Markup Language A graphical description of a simple XML document A graphical description of a simple XML document Status Published Starting year 1996 Authors W3C Basic SGML standards Related standards numerous among others: XHTML, RSS, Atom, KML ... Abbreviation XML Domain Worldwide web Website XML at W3C Portaal Portaalicoon Informatica Extensible Markup Language (XML) is a standard of the World Wide Web Consortium for the syntax of formal markup languages ​​with which you can display structured data in the form of plain text. This presentation is both machine-readable and readable for humans. The XML format is used to store data (as in the OpenDocument format) and to send data over the internet.

    Audio file formats by file extensions
        .aif - AIF audio file
        .cda - CD audio track file
        .mid or .midi - MIDI audio file.
        .mp3 - MP3 audio file
        .mpa - MPEG-2 audio file
        .ogg - Ogg Vorbis audio file
        .wav - WAV file
        .wma - WMA audio file
        .wpl - Windows Media Player playlist

    Compressed file extensions
        .7z - 7-Zip compressed file
        .arj - ARJ compressed file
        .deb - Debian software package file
        .pkg - Package file
        .rar - RAR file
        .rpm - Red Hat Package Manager
        .tar.gz - Tarball compressed file
        .z - Z compressed file
        .zip - Zip compressed file

    Disc and media file extensions
        .bin - Binary disc image
        .dmg - macOS X disk image 
        .iso - ISO disc image
        .toast - Toast disc image
        .vcd - Virtual CD

    Data and database file extensions
        .csv - Comma separated value file
        .dat - Data file
        .db or .dbf - Database file
        .log - Log file
        .mdb - Microsoft Access database file
        .sav - Save file (e.g., game save file)
        .sql - SQL database file
        .tar - Linux / Unix tarball file archive
        .xml - XML file

    Executable file extensions
        .apk - Android package file
        .bat - Batch file
        .bin - Binary file
        .cgi or .pl - Perl script file
        .com - MS-DOS command file
        .exe - Executable file
        .gadget - Windows gadget
        .jar - Java Archive file
        .py - Python file
        .wsf - Windows Script File

    
    Font file extensions
        .fnt - Windows font file

        .otf - Open type font file
        .ttf - TrueType font file

    Image file formats by file extension
        .ai - Adobe Illustrator file
        .bmp - Bitmap image

        .jpeg or .jpg - JPEG image

        .ps - PostScript file
        .psd - PSD image

        .tif or .tiff - TIFF image
